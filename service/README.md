# service

> A Koa2 project

## 文件结构 
├── bin  
│　　└── www.js　// 接口监听设置  
├── mysql　// 数据库  
│　　└── defaultConfig.js　// 数据库连接  
│　　└── mysqlConfig.js　// sql  
├── public   
├── routes  
│　　└── index.js // 静态文件路由  
│　　└── goods.js // 货品相关接口  
│　　└── records.js // 出入库相关接口  
│　　└── users.js // 账号相关接口  
├── views  
└── app.js // 主程序入口  

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8082
npm start
```
